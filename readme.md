### 插件简介

中国节日之春节，给网站加上漂亮的灯笼或者吉祥的福字，迎接一年一度的春节，弘扬中国传统文化。


### 使用方法

安装插件并激活，进入“中国节日”设置界面进行配置，可选：

- 显示灯笼，并设置灯笼上的字
- 显示福字，共有两种样式选择
